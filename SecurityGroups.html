Security Groups

  • Firewall for EC2 Instances
  • Regulate:
    1. Access to PORTS
    2. Authorise IP Ranges - IPv4 and IPv6
    3. Control of Inbound network
    4. Control of Outbound network
  • Can be attached to multiple EC2 Instances
  • Locked Down to Region
  • Live Outside of EC2 Instance
  • Time Out = Security Group Issue
  • Connection Refused = Application Error
  • Default Inbound: All traffic is BLOCKED
  • Default Outbound: All traffic is Authorized
